# Pipeline de Engenharia de Dados - Desafio Prático

Este projeto simula um fluxo de **ETL (Extract, Transform, Load)** focado em estruturar uma base de dados sólida (**Data Foundation**) para análise de negócios.

### O que este script faz:
- **Extração:** Lê dados brutos de vendas em formato CSV.
- **Transformação:** Limpeza de registros nulos e conversão de tipos de dados usando **Pandas**.
- **Regra de Negócio:** Cálculo de faturamento total para suporte a decisões estratégicas (**Impacto no DRE**).
- **Carga:** Gera um arquivo processado pronto para consumo em ferramentas de BI ou modelos de IA.

### Tecnologias:
- Python 3.x
- Pandas
